Processing details

I copy the output into a text file PG_review.txt, read it from ipython. json can parse the string into a dictionary with command 

import json; output = json.load(f)

Then I concert this dictionary by a third party package dicttoxml, which can be installed by 

sudo pip install dicttoxml

Converting command 

import dicttoxml; tree_str = dicttoxml.dicttoxml(output)

converts the dictionary into a xml string. Then I write it into a xml files. 

f = open('PG_output.xml', 'w')
f.write(tree_str)

The output xml file can be consulted through Chrome browser and if you want a human readable xml files in text editor, format the file by 

xmllint --format PG_output.xml > temp; mv temp PG_output.xml
