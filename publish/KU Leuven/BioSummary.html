1 Temperal Relation Extraction Biobliography Summary

The temporal relation extraction aims to extract the relations between the specific (task defined) events to the temperal anchors, which can be the document creation time (usually out of text) or the temporal descriptions mentioned in document. This task can be straightforwardly split into three parts: extraction of events, extraction of temporal description (Timex) and the extraction of the links between them. The tasks defined by TimeML, TempEval, etc. basically followed this principal.

The events and timexes are usually recognized independently using the methods used by NER problems (CRF, SVM, etc.).

1.1 Timex extraction

  • HeidelTime: High quality rule-based extraction and normalization of temporal expressions.
  • TRIPS and TRIOS sstem for TempEval-2: extracting temporal information from text. Deep semantic parsing (TRIPS) + logical pattern => events and timex. Classification (Makov Logic Network) => relation.
